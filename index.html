The best way to start developing

color_lens
計畫宗旨
本計畫定義為進入職場前的培育計畫，目標讓全台灣的學生都能夠參與此計畫，同時能夠解決業界人手不足，讓新一代的技術能夠萌芽、經歷得以傳承。
flash_on
計畫目標
我們認為，成為一個網站開發的自學者，不應該是一個人默默地前進。更多需要的是，一個資深的前輩/學長/助教能夠適時地提供指引與協助。我們透過資深工程師帶領入門新手，讓熱情的學員、初學者也可以踏入技術的殿堂，達到技術的傳承與知識的延續。
build
可以學得技術
開發：Git / UNIX 系統操作 / 開發環境配置 基礎：HTML / JavaScript / CSS / Node.js / Express / MySQL 進階：Passport / Security / React / Redux / MongoDB / Testing
第一階段
  1. 報名 KKTIX（注意：需註冊 KKTIX 帳號，後續才能進行修改）
  2. 開始練習 freeCodeCamp 線上課程
    • HTML5 and CSS
    • Basic JavaScript
    • Git
    • Node.js and Express.js
  3. 加入社團，遇到問題可以在社團裡面討論
  4. 完成練習，更新 kktix 資料
  5. 等待第二階段通知
